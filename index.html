Planos de Aula - Iniciação Digital

Nível Básico

Aula 1: Conhecendo o Computador

Objetivo: Identificar componentes e ligar/desligar o computador.

Passo a Passo:

  • Reúna os alunos em torno de um computador (5 min).
  • Aponte monitor, CPU, teclado, mouse e nomeie (10 min).
  • Explique funções: "Mouse move, teclado digita" (5 min).
  • Divida em duplas para ligar o computador (15 min).
  • Abra o sistema operacional (ex.: Windows) (10 min).
  • Desligar pelo menu "iniciar" > "desligar" (5 min).

Tarefa Extra (se sobrar tempo): Explorar a Área de Trabalho

  • Mostre ícones: "Este é o Paint, este é o Chrome" (5 min).
  • Peça para abrirem 2 programas diferentes (10 min).
  • Oriente a fecharem os programas (5 min).

Ferramentas: Computador, projetor (opcional).

Avaliação: Ligar/desligar sem ajuda; (extra) abrir e fechar programas.

Aula 2: Usando o Mouse e o Teclado

Objetivo: Aprender movimentos básicos.

Passo a Passo:

  • Mostre clique esquerdo, direito e rolagem (10 min).
  • Demonstre no Paint: abrir e desenhar (10 min).
  • Peça para abrirem o Paint (10 min).
  • Oriente a desenhar linha e círculo (10 min).
  • Digitar "oi" com teclado (10 min).

Tarefa Extra (se sobrar tempo): Desenho Livre no Paint

  • Oriente a escolherem uma cor diferente (5 min).
  • Peça para desenharem algo simples (ex.: casa) (10 min).
  • Salve o desenho na área de trabalho (5 min).

Ferramentas: Computador, Paint.

Avaliação: Linha desenhada e "oi" digitado; (extra) desenho salvo.

Aula 3: O que é a Internet?

Objetivo: Entender o conceito básico.

Passo a Passo:

  • Explique: "Internet é uma biblioteca gigante" (10 min).
  • Mostre vídeo no YouTube (10 min).
  • Abra o Chrome (5 min).
  • Digite "www.google.com" (5 min).
  • Duplas abrem Chrome e acessam Google (20 min).

Tarefa Extra (se sobrar tempo): Explorar o Google Imagens

  • No Google, clique na aba "Imagens" (5 min).
  • Pesquise "paisagens" e mostre (5 min).
  • Peça para clicarem em uma imagem e voltarem (10 min).

Ferramentas: Computador, Chrome, YouTube.

Avaliação: Abrir navegador; (extra) navegar nas imagens.

Aula 4: Primeiros Passos na Navegação

Objetivo: Usar um navegador básico.

Passo a Passo:

  • Mostre barra de endereços (5 min).
  • Digite "www.nasa.gov" (5 min).
  • Explique "voltar" e "avançar" (5 min).
  • Peça para digitarem "www.nasa.gov" (15 min).
  • Clicar em "voltar" e "avançar" (15 min).

Tarefa Extra (se sobrar tempo): Explorar um Site Infantil

  • Digite "www.clubedopequenoexplorador.com.br" (5 min).
  • Peça para clicarem em uma seção (ex.: jogos) (10 min).
  • Oriente a voltarem à página inicial (5 min).

Ferramentas: Computador, Chrome.

Avaliação: Navegar entre páginas; (extra) explorar site.

Aula 5: Comportamento Online - Primeiras Regras

Objetivo: Conhecer regras básicas de comportamento.

Passo a Passo:

  • Pergunte: "Como se comportam com amigos?" (5 min).
  • Explique: não gritar (usar CAPS), ser educado (10 min).
  • Mostre exemplos: "OI!!!" vs. "Oi, tudo bem?" (5 min).
  • Crie um chat fictício (ex.: Google Docs) (5 min).
  • Peça para escreverem: "Olá, gostei da aula" (15 min).
  • Leia mensagens e comente (10 min).

Tarefa Extra (se sobrar tempo): Responder a um Colega

  • Mostre mensagem: "Oi, tudo bem?" (5 min).
  • Peça para responderem: "Oi, sim, e contigo?" (10 min).
  • Leia e comente (5 min).

Ferramentas: Computador, Google Docs.

Avaliação: Mensagem segue regras; (extra) resposta educada.

  • Peça para identificarem o seguro (15 min).

Tarefa Extra (se sobrar tempo): Criar um Link Fictício

  • Peça para imaginarem link perigoso (5 min).
  • Escrevam no Docs (10 min).
  • Discuta: "Por que parece perigoso?" (5 min).

Ferramentas: Computador, slides, Google Docs.

Avaliação: Identificação correta; (extra) link criado.
